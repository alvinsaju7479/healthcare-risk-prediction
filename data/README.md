\# Data



This project uses the \*\*Diabetes 130-US hospitals readmission dataset\*\*.



\## How to get the data

Download from Kaggle (search: \*Diabetes 130 US hospitals readmission\*).



Place the CSV here:

\- `data/raw/diabetic\_data.csv`



The pipeline will generate:

\- `data/processed/processed.csv`

\- `data/processed/train.csv`, `val.csv`, `test.csv`



⚠️ Data is not included in this repository.
